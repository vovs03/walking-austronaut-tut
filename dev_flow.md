# Dev_flow

## Create Animations in Atlas-file

> `.\main\austronaut.atlas`

1. Add into Atlas `Animation Group` :
	- left
	- right
	- back
	- front
2. Set images inside each group.
3. Set Speed `From 60 to 12`
